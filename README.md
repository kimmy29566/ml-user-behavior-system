# ML User Behavior Analytics System

ระบบวิเคราะห์พฤติกรรมผู้ใช้งานเว็บไซต์ด้วย Machine Learning

## Features
- Real-time Analytics
- Machine Learning Prediction
- Dashboard Visualization
- User Tracking

## Tech Stack
- React.js
- FastAPI
- PostgreSQL
- scikit-learn

## Run Project

### Frontend
npm run dev

### Backend
uvicorn app.main:app --reload
